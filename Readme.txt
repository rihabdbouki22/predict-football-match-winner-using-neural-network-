Project Context

This project was done by Rihab Dbouki and Mohamed Kacem Ferchichi as school projet in the 'Project ISID' lesson.
It aims to predict the football match winner based on LSTM (Long short Time Memory).

Dataset

The dataset used here has been obtained from football-data.co.uk. Datasets of the English Premier league have been taken from seasons 2010-11 to 2016-17.

Data preparation
The file datacleaning.ipynb takes the raw data from the website and add attributes regarding the win streaks and the loss streaks 
for every team. Also, eplStandings.csv contains the final ranks of all the teams in the English Premier League from 2010-11 to 2016-17.
